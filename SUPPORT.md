# Support for Deadline Guardian (AURA)

Thank you for using AURA! If you encounter issues or have questions, please use the following channels to find support.

---

## 🔍 Self-Help & Debugging

Before reaching out, check if your problem can be solved with these quick checks:

1.  **Check API Key Configuration**: AURA requires a valid Google AI Studio key. If you see errors about "API Key Missing", double-check that your `.env` file matches `.env.example` exactly and that you restarted your dev server.
2.  **Verify Node Version**: Make sure you are using Node.js v18 or higher (`node -v`).
3.  **Clean Dependencies**: If you see compilation or module resolution errors, try cleaning your node cache and re-installing:
    ```bash
    rm -rf node_modules package-lock.json
    npm install
    ```
